# Toy Minecraft Shaders

Toy Minecraft Shaders 是一个简单的基于Optifine mod的shader集，包括复数渲染算法及效果。

# 安装到 MC

1. 首先需要安装OptifineHD mod
2. 然后将着色器包文件复制到游戏目录`.minecraft/shaderpacks`
3. 在游戏菜单中选择：`vedio setting --> shades`
